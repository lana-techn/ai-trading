### **The 2% Rule**
- **Never risk more than 2%** of your total capital on a single trade
- If you have $10,000, maximum loss per trade = $200
- This allows for 50 consecutive losses before account depletion

### **Position Sizing Formula**

### **Stop Loss Strategy**
- **Always set stop losses** before entering trades
- **Technical stops**: Based on support/resistance levels
- **Percentage stops**: Fixed 3-5% below entry
- **ATR stops**: Based on Average True Range volatility
